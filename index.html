PassKeep Pwd Manager

PassKeep is a CLI based password manager built with python which stores encrypted passwords using the MySQL database. It is able to save new passwords, edit and view existing passwords and access all of them with a single master password. It's a simple project that I made in High School. I learnt integration and initialization of MySQL Databases in a python project, encryption and authentication of passwords, and efficient code management and documentation.
